# Wordle Suggestion Engine
## General Info
Have you ever found youself in a position where you do not know what possible words exits while playing wordle?

This userscipt will give you a suggested word that contains the green and yellow letters. 

Important: This is NOT an optimized engine rather is meant to randomly pick a word from a list of all possible words that contain the green and yellow letters. 

Warning: this ruins the fun of wordle.

## How to Use
Using `TamperMonkey` extension in your browser:
- create new script
- copy the code in `wordleSE.js` and paste it in your new script
- press `ctrl + s`

The script will automatically run. 


## Caution
This script is a result of an afternoon project. I am NOT a JS developer. Thus please use with caution and should you find any issue please report it. 

## Tips
To optimize your Wordle playing you can choose "SOARE" and "CLINT" as your first two words and the engine is very likely to find the correct wordle as the 3rd guess.
